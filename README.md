# 🌊 رصين — Raseen

> منصة ذكاء اصطناعي سعودية للتنبؤ بالاحتراق الوظيفي عبر تحليل البصمة الرقمية للأداء.

## 📁 هيكل المشروع

```
raseen/
├── index.html         ← Router (يعيد التوجيه حسب الدور)
├── login.html         ← صفحة تسجيل الدخول مع اختيار الدور
├── user.html          ← لوحة المستخدم البسيطة
├── employee.html      ← لوحة الموظف (الكاملة)
├── manager.html       ← لوحة المدير (Admin Dashboard)
├── shared.css         ← التصميم المشترك (Marble + Glass + Theme)
├── shared.js          ← Auth + Theme + Utilities
├── api/
│   └── chat.js        ← Vercel Serverless function
├── vercel.json        ← إعدادات النشر
├── package.json
├── .gitignore
└── .env.example
```

## 🔐 نظام تسجيل الدخول

عند فتح الموقع، يتم التوجيه التلقائي حسب الحالة:

- **غير مسجّل** → `login.html`
- **مسجّل كمستخدم** → `user.html`
- **مسجّل كموظف** → `employee.html`
- **مسجّل كمدير** → `manager.html`

كل صفحة تتحقق من الدور وتُعيد التوجيه إذا كان غير مناسب.

## 👥 الأدوار الثلاثة

### 👤 المستخدم (User)
عرض بسيط — معلومات عن المنصة، تمارين الاسترخاء، موارد عامة.

### 💼 الموظف (Employee)
لوحة كاملة:
- 📊 الأداء (طاقة عقلية، تحليل أسبوعي، إنجازات صغيرة)
- ✅ المهام (مع subtasks، AI sorting، difficulty)
- 💚 مركز الدعم (تنفّس، تأريض، HR، مكتبة)
- 🔒 الخصوصية (تكاملات، إعدادات)

### 👔 المدير (Manager)
لوحة إدارية:
- 📊 نظرة عامة (stats، اتجاهات، تنبيهات، أقسام)
- 👥 إدارة المستخدمين (بحث، إضافة، تعديل، حذف)
- 💼 إدارة الموظفين (مؤشرات الرفاهية، الطاقة، الخطر)
- 📈 التقارير والتحليلات (Charts متعددة)

## ✨ المميزات المشتركة

- 🌊 خلفية رخامية مائية متحركة
- 🌗 وضع داكن/مضيء (Dark/Light mode)
- 🌐 RTL كامل بخط IBM Plex Sans Arabic
- 📱 Responsive لكل الأحجام
- 🎨 Glassmorphism & Ocean theme متسق

## 🚀 النشر على Vercel

```bash
npm install -g vercel
vercel login
vercel --prod
```

أو اقرأ `AGENT.md` لاستخدام AI agent للنشر التلقائي.

## 🔑 إضافة API Key (للمستشار الذكي)

```bash
vercel env add ANTHROPIC_API_KEY production
vercel --prod
```

## 🧪 الاختبار المحلي

افتح `index.html` مباشرة في المتصفح، أو:
```bash
npx serve .
```

### بيانات تجريبية للتسجيل:
- البريد: أي بريد صالح (مثلاً `test@raseen.sa`)
- كلمة المرور: ٤ أحرف على الأقل
- اختر أي دور (مستخدم / موظف / مدير)
